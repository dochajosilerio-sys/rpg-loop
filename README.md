# ⚔️ RPG de Loop Infinito

Um jogo de console desenvolvido em Python que explora conceitos de **estruturas de repetição (while)** e **lógica de progressão incremental**.

## 🕹️ Funcionalidades
- **Sistema de Batalha Automático:** O herói enfrenta monstros que ficam mais fortes a cada ciclo.
- **Progressão de Atributos:** A cada vitória, o herói ganha XP, aumenta seu ataque e recupera uma porção de vida.
- **Esconamento de Dificuldade:** A vida e o dano dos monstros crescem proporcionalmente ao número de loops.

## 💻 Tecnologias
- **Linguagem:** Python 3.x
- **Execução:** Terminal / Termux

## 🚀 Como Jogar
Basta executar o script e inserir o nome do seu herói:
```bash
